## Overview

### Description

Messenger App is a mobile communication application that allows users to send and receive real-time text messages with friends.
Users can create an account, start conversations, and receive push notifications when new messages arrive.
The goal of the app is to provide a simple, fast, and reliable way for users to stay connected anytime and anywhere.

### App Evaluation

[Evaluation of your app across the following attributes]
- **Category:** Social / Communication
- **Mobile:** This app uses mobile features such as push notifications, camera for profile pictures, and real-time messaging which fits mobile usage perfectly.
- **Story:** People communicate daily through messaging apps. This app helps users stay connected with friends, classmates, and family members instantly.
- **Market:** Messaging apps have a very large user base including students, professionals, and families.
- **Habit:** Users will open the app frequently to check messages and reply to conversations, making it highly habit-forming.
- **Scope:** The basic version will include text messaging and conversation list. Future versions may include voice messages, image sharing, and group chats.
